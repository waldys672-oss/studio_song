# سمو — Premium Arabic Audio Service Website

Build a luxurious, mobile-first, RTL Arabic website for a Gulf audio brand specializing in زفات, شيلات, and custom audio content. The site targets Saudi/Gulf audiences and is conversion-focused.

## Architecture

**Static multi-page site** using vanilla HTML, CSS, and JavaScript — no framework needed. Each page is a standalone `.html` file sharing a common CSS design system and JS utilities.

### File Structure

```
studio_song/
├── index.html              # Homepage
├── category.html            # Category page (dynamic via query param)
├── custom-order.html        # Custom request form page
├── about.html               # About page
├── css/
│   ├── variables.css        # Design tokens (colors, fonts, spacing)
│   ├── base.css             # Reset, RTL, typography, global styles
│   ├── components.css       # Buttons, cards, navigation, footer
│   └── pages.css            # Page-specific styles
├── js/
│   ├── data.js              # Sample works data, categories, artists
│   ├── components.js        # Shared components (nav, footer, WhatsApp)
│   └── main.js              # Page-specific logic, routing, interactions
└── assets/
    └── (generated images)
```

## Design System

### Color Palette
| Token | Value | Usage |
|-------|-------|-------|
| `--gold` | `#D4AF37` | Primary accent, CTAs, highlights |
| `--gold-light` | `#F5E6A3` | Hover states, subtle accents |
| `--black` | `#0A0A0A` | Primary background |
| `--black-soft` | `#1A1A1A` | Card backgrounds, elevated surfaces |
| `--black-medium` | `#2A2A2A` | Borders, dividers |
| `--white` | `#FAFAFA` | Primary text |
| `--white-muted` | `#B0B0B0` | Secondary text |
| `--gradient-gold` | `linear-gradient(135deg, #D4AF37, #B8960C)` | Premium buttons |
| `--gradient-hero` | `radial-gradient(...)` | Hero background |

### Typography
- **Primary Font**: `Tajawal` (Google Fonts — clean, modern Arabic font)
- **Display Font**: `Amiri` (for ornamental headings, optional)
- Font sizes: Fluid typography with `clamp()` for responsive scaling

### Spacing & Layout
- 8px base grid system
- Container max-width: `1200px`
- Mobile-first breakpoints: `480px`, `768px`, `1024px`, `1280px`

## User Review Required

> [!IMPORTANT]
> **Video/Audio Samples**: The site will use placeholder video embeds (YouTube-style iframes or styled `<video>` elements with poster images). You'll need to replace these with your actual content URLs later.

> [!IMPORTANT]
> **WhatsApp Number**: The WhatsApp button and form submission will use a placeholder number (`+966XXXXXXXXX`). You'll need to update this with your real WhatsApp Business number.

> [!NOTE]
> **No Backend**: The custom order form will redirect to WhatsApp with pre-filled message data. No server-side processing is included. If you need a backend (email, database), that would be a separate phase.
